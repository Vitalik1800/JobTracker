# 🚀 JobTracker

Track your career journey.

JobTracker is a modern Android application for managing job applications, tracking interview progress, monitoring statistics, and exporting data.

---

## ✨ Features

### 📋 Job Management

* Add vacancies
* Edit vacancies
* Delete vacancies
* View job details

### 🔍 Search & Filtering

* Search by title or company
* Filter by status
* Sort by:

  * Newest
  * Oldest
  * Title
  * Company

### ⭐ Favorites

* Add jobs to favorites
* Separate favorites screen

### 📊 Statistics Dashboard

* Total jobs
* Favorite jobs
* Applied jobs
* Interviews
* Technical interviews
* Offers
* Rejected jobs
* Active jobs
* Success rate
* Applied → Interview conversion
* Interview → Offer conversion
* Pie Chart visualization

### 📂 Export & Backup

* Export to PDF
* Export to CSV
* Share exported files
* Database backup

### 🎨 UI

* Material 3
* Dark Theme
* Splash Screen
* Responsive layouts

---
